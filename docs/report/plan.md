## 一、项目概述

### 1. 项目背景
当前校园与社区心理咨询多采用线下登记、人工排期，存在效率低、时间易冲突、记录难管理等问题。为优化服务流程、提升咨询体验，开发一套线上心理健康预约系统，实现线上预约、智能排期、冲突自动检测，满足校园心理中心与社区服务站的日常管理需求。

### 2. 项目目标
1. 实现用户注册登录、心理咨询在线预约。
2. 支持咨询师可预约时段设置与日程管理。
3. 实现预约时间冲突自动检测与提示。
4. 提供预约查询、取消、修改等功能。
5. 系统稳定易用，支持校园与社区双场景使用。

### 3. 开发环境与条件
- 开发语言：**Java**
- 后端框架：**Spring Boot 3**
- 前端框架：Vue3 + Element-Plus + Axios
- 数据库：MySQL

---

## 二、需求分析

### 1. 目标用户与使用场景
- 普通用户：学生/社区居民，在线预约、查看记录。
- 咨询师：设置排班、管理预约日程。
- 管理员：用户管理、咨询师管理、数据统计。
- 使用场景：校园心理中心、社区服务站线上预约服务。

### 2. 核心功能清单
1. 用户注册、登录、个人信息管理
2. 咨询师信息展示与可约时间查看
3. 预约提交与**时间冲突自动检测**
4. 预约取消、修改、状态提醒
5. 咨询师日程排班与管理
6. 管理员后台统一管理

### 3. 非功能需求
- 性能：页面响应＜2秒，支持多人同时在线。
- 安全：密码加密、权限控制、防止越权操作。
- 兼容性：支持PC端主流浏览器，移动端适配。
- 易用性：3步内完成预约，操作简单直观。

---

## 三、技术路线

### 1. 总体架构
前后端分离架构：
- 前端：Vue3 负责页面渲染与交互
- 后端：Spring Boot 3 提供 RESTful 接口
- 数据库：MySQL 存储业务数据
- 通信：Axios 请求 + JSON 数据交互

### 2. 主要技术
- 后端：Java 17、Spring Boot 3、Spring MVC、MyBatis
- 前端：Vue3、Vite、Element Plus、Axios
- 数据库：MySQL 8.0
- 工具：Maven、Git、Postman

### 3. 数据设计初步考虑
- 主要实体：用户、咨询师、预约信息、日程安排
- 关键数据表：
  - user（用户表）
  - counselor（咨询师表）
  - schedule（日程表）
  - appointment（预约表）
- 核心关系：用户与预约多对多；咨询师与日程一对多。
