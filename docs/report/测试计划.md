# 测试计划

## 一、测试概述
通过组员在项目功能实现过程中发现问题，由专人进行全面测试与 Bug 修复，确保系统功能正常运行。

## 二、测试说明
其他组员完成代码编写后，运行并检查实际效果，发现如图片未显示、部分功能缺失等情况，由测试人员重新全面检查代码并修正，覆盖前端到后端。
出现问题则进行修复，无问题则保持原状不变。
